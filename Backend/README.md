# Project YvY Backend

## Running the app.

#### Requirements
 ```bash
  python -v 3.6.12
  ```

#### Install dependencies
 ```bash
  pip install Flask
  pip install cloudant
  pip install Flask-Cors
  pip install cos-aspera
  pip install ibm-cos-sdk
  pip install ibm-cos-sdk-core
  pip install ibm-cos-sdk-s3transfer
  ```

#### Run development mode
 ```bash
  python main.py
  ```
